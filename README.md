# OSINT Toolkit Dashboard

A modern cybersecurity-inspired **OSINT (Open Source Intelligence) Toolkit Dashboard** built using **HTML, CSS, JavaScript, Node.js, and Express.js**.

This project performs real-time **WHOIS** and **DNS reconnaissance** operations through a clean futuristic dashboard interface.

---

# Features

## WHOIS Lookup
- Domain registration lookup
- Registrar information
- Domain creation date
- Country information
- Invalid domain handling

## DNS Lookup
Supports multiple DNS record types:
- A Records
- MX Records
- TXT Records
- NS Records
- CNAME Records

## UI Features
- Cybersecurity-themed interface
- Real-time API integration
- Dynamic data rendering
- Responsive design
- Live backend communication
